01 / Hero Section

Hero — первое впечатление

Определяет конверсию. Должен работать за 3 секунды: кто вы, что делаете, почему вам доверять. Ниже — лучшие примеры и что забрать под себя.

stripe.com

Gradient silk background + gradient text fade

Розово-оранжевый шёлковый градиент занимает правую часть hero. Заголовок ~56px с эффектом text gradient fade (часть текста серая, часть чёрная — создаёт глубину). Сразу живая метрика сверху ("1.62% Global GDP"). Dual CTA: синий filled + белый outlined. Лого-бар клиентов (Amazon, Google, NVIDIA) прямо под hero без разрыва.

Что берём для elfprint.ru
  • Lime-gradient (#C6F24E → transparent) как фоновое пятно за hero-фото
  • Live-метрика сверху: "20+ лет на рынке" pill badge
  • Лого-бар клиентов: Армия России, ЛДПР, Летуаль, Тинькофф, Яндекс, Лукойл
  • Dual CTA: "Обсудить проект" (lime filled) + "Смотреть каталог" (outlined)
apple.com

Центрированный hero: продукт — звезда

Название продукта (~64px bold) → слоган в 1 строку → dual CTA (Learn more + Buy) → фото продукта снизу, держит на руке. Фон — тёплый серый #F5F5F7, не чистый белый. Абсолютный минимум текста. Продукт — единственный визуальный акцент.

Что берём для elfprint.ru
  • Layout: заголовок → подзаголовок → chips → CTA → фото мерча
  • Фон #F5F5F7 (уже используем) — это правильно
  • Продукт на переднем плане: худи/футболка с логотипом клиента
  • Принцип "меньше текста = сильнее впечатление"
vercel.com

Announcement pill + grid background + geometric art

Сверху — pill badge с анонсом ("Events → Ship 26 is coming"). Центрированный заголовок ~48px. Подзаголовок серым. Dual CTA (чёрный filled + белый outlined). Тонкие grid-линии на фоне. Абстрактная геометрическая визуализация снизу — цветовой акцент без фотографии.

Что берём для elfprint.ru
  • Announcement pill: "Ответ за 30 мин" или "Акция: скидка на первый заказ"
  • Subtle grid-pattern на фоне hero (CSS-only, без изображений)
  • Если нет фото — использовать абстрактную геометрию с lime-акцентом
fujida.pro

Гигантская типографика = визуал

Название бренда занимает весь viewport как графический элемент. Чёрный на белом. Подзаголовок деликатно рядом с последней буквой. "Крутите ↓" справа как scroll hint. Никаких фотографий — только шрифт. Российский сайт, отлично работает.

Что берём для elfprint.ru
  • Масштаб: заголовок 56-72px minimum, вес 800
  • Scroll-hint элемент "Листайте ↓"
  • На странице "О компании" — можно сделать hero с гигантским "ОГНИ ЭЛЬФОВ"
02 / Services & Features

Bento Grid — секция услуг

Вместо одинаковых карточек 3-в-ряд — разноразмерная сетка с визуалами. Выглядит дороже и интереснее.

stripe.com

Bento Grid — разноразмерные карточки услуг

Верхний ряд: 2 карточки (широкая 2/3 + узкая 1/3). Нижний ряд: 3 равные. Внутри каждой — заголовок + визуал (скриншот продукта, UI-элемент, график). Карточки с border-radius ~16px, subtle border, hover-эффект. Иконка "expand" в углу каждой карточки.

Что берём для elfprint.ru
  • Grid: 2 колонки сверху (DTF-печать широкая + Шелкография узкая), 3 снизу (Вышивка, УФ-печать, Сублимация)
  • Внутри карточек: фото процесса / результата + название технологии + "от X руб."
  • Hover: карточка поднимается (translateY) + акцентный border
  • Corner icon: стрелка "перейти" для каждой услуги
03 / Trust & Social Proof

Доверие — числа и лого

Крупные метрики + логотипы клиентов + рейтинги. Работает на B2B-аудиторию сильнее любого текста.

stripe.com — trust bar

Client logo bar + live metrics

Лого-бар идёт горизонтальной полосой сразу после hero. Серые логотипы на белом фоне. Выше — живая метрика ("1.62% Global GDP running on Stripe") с анимированным счётчиком. Лого автоматически скроллятся если не помещаются.

Что берём для elfprint.ru
  • Лого-бар клиентов: серые SVG логотипы, горизонтальный scroll на мобильных
  • Stats-секция: 20+ лет | 2 000+ клиентов | 6 технологий | 300+ заказов/мес
  • Count-up анимация при scroll (IntersectionObserver + requestAnimationFrame)
  • Яндекс.Отзывы виджет с реальным рейтингом 4.9
04 / CTA Section

Тёмная CTA-секция

Контраст с основным светлым фоном. Привлекает внимание к форме.

linear.app

Dark background + высококонтрастная типографика

Чёрный/тёмно-серый фон (#0A0A0A). Белый текст крупным кеглем. Pill badge справа с акцентным сообщением. Product screenshot ниже. Минимум элементов — максимум воздуха.

Что берём для elfprint.ru
  • CTA-секция: тёмный фон #1D1D1F → белый заголовок "Обсудим ваш проект?"
  • Lime CTA кнопка (#C6F24E) на тёмном = максимальный контраст
  • Форма inline (name + phone + button) — уже есть, но добавить glow-эффект
  • Мессенджеры TG/MAX как secondary action
05 / Brand Atmosphere

Атмосфера бренда

Full-bleed изображения, иммерсивные первые экраны, продукт = lifestyle.

cocosbeauty.shop

Full-bleed фото + overlay типографика

Фото на весь экран. Название бренда поверх белым outline-шрифтом. Слоган внизу. Scroll-hint (мышка). Соцсети сбоку. Тёмная атмосфера = премиум. Российский сайт, работает хорошо.

Что берём для elfprint.ru
  • Для страницы "О компании" — full-bleed фото фабрики/команды
  • Для portfolio — full-width кейсы с overlay текстом
  • Scroll-hint элемент (анимированная мышка/стрелка)
