# UX-анализ сайтов конкурентов в нише мерча/промо-продукции

**Дата анализа:** 19 марта 2026
**Методология:** Визуальный UX-аудит через Playwright (DOM-snapshot + скриншоты), критический анализ с позиции топ-дизайнера и маркетолога.

## 1. PRINTBAR.RU — B2C принт-он-деманд

### Hero-секция
- **Заголовок:** "Футболки с принтами для всех - Интернет магазин Print Bar!" — банальный, нулевая эмоция, нет УТП. Просто констатация факта.
- **CTA:** Отсутствует в hero-секции как таковой. Есть логотип PRINT BAR крупно, иконки "Придумано в России / Полная запечатка 360 / 72 часа" — но это не CTA, а информационные блоки.
- **Визуал:** Два модельных фото (мужчина и женщина) на розово-фиолетовом фоне. Фотографии среднего качества, композиция перегружена.
- **КРИТИКА:** Hero выглядит как баннер из 2018-2019 года. Слишком много элементов конкурируют за внимание. Нет четкого действия, которое пользователь должен совершить. Шрифт Bebas Neue не загружается (ошибка woff2 в консоли!) — критический технический баг, влияющий на типографику.

### Навигация
- Двойная шапка: верхняя (Каталог / Свой дизайн / Поиск / Корзина) + подшапка (Новинки / Конструктор одежды / ОПТ 70% / Избранное / Где мой заказ?).
- Мегаменю на hover с категориями: Мужчинам / Женщинам / Детям — нормальная структура, подкатегории одежды.
- **КРИТИКА:** Информационный шум. "ОПТ 70%" — красным — выглядит как spam. "ГДЕ МОЙ ЗАКАЗ?" — красным, отвлекает от основного пути пользователя. Два уровня навигации создают когнитивную перегрузку. Подшапка дублирует функционал.

### Карточки товаров
- Сетка 4 колонки. Товар показан на белом фоне (фото самой футболки без модели).
- Цена: зачеркнутая старая + новая (все -40%). Иконка "ХЛОПОК" внизу каждой карточки.
- Есть избранное (сердечко), бейдж скидки.
- **КРИТИКА:** ВСЕ товары со скидкой -40% — это обман доверия, "вечная скидка" уничтожает ценность. Карточки примитивные — нет hover-эффекта со вторым фото, нет кнопки "В корзину", нет размеров. Названия товаров обрезаются ("Барселона Форма Новая Госте..."). Фотографии — просто развертка футболки, не лайфстайл. Внизу главной страницы — бесконечный список текстовых ссылок на тематики (Лошади, Совы, Бабочки...) — это SEO-мусор, вредящий UX.

### Визуальное качество
- **Типографика:** Проблемная — Bebas Neue не загружается, fallback-шрифт ломает дизайн. Тексты мелкие.
- **Цветовая палитра:** Красный + синий + белый. Неконсистентность: красный логотип, синие кнопки, зеленые иконки хлопка.
- **Whitespace:** Недостаточно. Страница перегружена.
- **Уровень: УСТАРЕВШИЙ (2018-2019).** Визуально отстал на 6-7 лет.

### Доверие и конверсия
- Social proof: Нет на главной. Нет отзывов, нет цифр, нет логотипов клиентов.
- Контакты: Только email в футере, соцсети (VK, Twitter, WhatsApp, Telegram, Viber).
- Нет телефона на видном месте.
- **КРИТИКА:** Полное отсутствие элементов доверия. Нет гарантий, нет процесса заказа на главной. Путь пользователя неочевиден. JS-ошибки в консоли (VK не определен, Mindbox tracker падает) — техническая запущенность.

### Оценки Printbar.ru
| Критерий | Оценка |
|---|---|
| Дизайн | 3/10 |
| UX | 4/10 |
| Конверсионная архитектура | 3/10 |
| Mobile | 4/10 |

## 2. PRINTIO.RU — B2C принт-он-деманд

### Hero-секция
- **Баннер:** "ЗАКАЖИ ПРОФЕССИОНАЛЬНЫЙ ДИЗАЙН" + кнопка "Заказать" — красный баннер с иллюстрацией дизайнера. Слайдер с несколькими баннерами.
- **Подзаголовок:** "Футболки с принтами, одежда, подарки на заказ" — ниже, отдельно. Нормальная формулировка, но без эмоции.
- **CTA:** "Заказать" на баннере — маленький, теряется.
- **КРИТИКА:** Hero-секция слабая. Баннер выглядит как рекламная вставка, а не как основное ценностное предложение. Нет единого мощного заголовка. Иллюстрация в стиле Undraw/generic — не уникальная.

### Навигация
- Верхняя строка: город / телефон 8-800 / Помощь дизайнера / Где мой заказ / Контакты / Доставка / Гарантии / Войти / Регистрация — ДЕВЯТЬ пунктов в одну строку.
- Основная: Принтио (лого) / Свой дизайн / Готовые дизайны / Подобрать подарок / Все для бизнеса / Поиск / Корзина.
- **КРИТИКА:** Верхняя навигация перегружена — 9 пунктов мелким шрифтом. Это 2015-й подход. Нет мегаменю. Нет категорий товаров в основной навигации — нужно угадывать, что "Готовые дизайны" это каталог.

### Карточки товаров (главная)
- Категории представлены как карточки-баннеры с фото: "Онлайн-редактор", "Термонаклейки", "Варежки", "Футболки", "Носки", "Свитшоты".
- На каждой карточке — реальное фото продукта + 1-2 кнопки ("ВЫБРАТЬ" / "СОЗДАТЬ").
- **КРИТИКА:** Карточки категорий неоднородны — разный стиль фото, разные CTA. "Онлайн-редактор" и "Термонаклейки" — это не одного уровня сущности. Смешивание сервисов с товарами сбивает. Фото с эмодзи в заголовках (👕, 🧤) в title — несерьезно для бизнеса.

### Визуальное качество
- **Типографика:** Чистая, читаемая. Serif-лого "Принтио" — приятный бренд.
- **Цветовая палитра:** Белый + красный акцент. Минималистично.
- **Whitespace:** Достаточно, страница дышит.
- **Уровень: СРЕДНИЙ.** Чище чем Printbar, но без претензии на современность.

### Доверие и конверсия
- Телефон 8-800 на видном месте — хорошо.
- "Помощь дизайнера" — хороший сервисный элемент.
- Чат "Задать вопрос" — присутствует.
- Нет отзывов на главной, нет логотипов клиентов.
- **КРИТИКА:** Сервис "Помощь дизайнера" — сильная фишка, но она спрятана в мелкой навигации. Нет социального доказательства. Процесс заказа неочевиден.

### Оценки Printio.ru
| Критерий | Оценка |
|---|---|
| Дизайн | 5/10 |
| UX | 5/10 |
| Конверсионная архитектура | 4/10 |
| Mobile | 5/10 |

## 3. VSEMAYKI.RU — B2C лидер рынка

### Hero-секция
- **Баннер-слайдер:** "Готовимся к теплу — До -30% к весне!" — актуальная акция, CTA "Перейти в каталог".
- Второй слайд: "Сделай свой уникальный принт!" — ссылка на конструктор.
- Ниже — горизонтальный скролл коллекций: "Русский стиль", "Очень странные дела", "Tiny Bunny", "Геймерам", "Мемы", "Аниме".
- **КРИТИКА:** Баннер визуально приемлемый, но дизайн рукописного шрифта "Готовимся к теплу" — немного дешево. CTA "ПЕРЕЙТИ В КАТАЛОГ" — мелкий, плохо контрастирует с фоном баннера. Коллекции-баннеры ниже — хороший паттерн, но визуальное качество иллюстраций неоднородно.

### Навигация
- Верхняя строка: Город / Покупателям / Партнерская программа / Где мой заказ?
- Основная: Лого / Каталог / Свой дизайн / Опт / Поиск ("по 110 000 дизайнов") / Профиль / Просмотренные / Избранное / Корзина.
- Подкатегории: Мужчинам / Женщинам / Детям / Аксессуары / Для дома / Подарки / Школа / SALE — 8 пунктов.
- **КРИТИКА:** Навигация перегружена, но функциональна. 8 категорий — на грани допустимого. "Школа" — спорная категория в основном меню. Иконки в header (профиль, просмотренные, избранное, корзина) — 4 иконки без подписей — не самая очевидная навигация.

### Карточки товаров
- Двойное фото с hover (товар на модели + второй ракурс) — ХОРОШО.
- Цена с акцией: "Цена 860 Р" + зачеркнутая + бейдж "-30%".
- Название товара под фото, подробное.
- **КРИТИКА:** Карточки функциональные, но дизайн простоватый. Нет кнопки "В корзину" при hover — пропущенный конверсионный элемент. Цветовые варианты не показаны. Однообразная сетка без визуальных акцентов. Длинные названия товаров ("Мужская футболка oversize 3D Велес славянский Бог Богатырь") — обрезаются.

### Визуальное качество
- **Типографика:** Лого VSEMAYKI яркое (красно-фиолетовый градиент). Тексты читаемые.
- **Цветовая палитра:** Белый фон + желтые/фиолетовые акценты. Живая, молодежная.
- **Whitespace:** Нормально, но страница длинная — много контента.
- **Уровень: СРЕДНИЙ+.** Рабочий e-commerce, но без претензии на премиальность или дизайнерский уровень.

### Доверие и конверсия
- В футере: блоки "Гарантия", "Доставка", "Сервис" — с иконками и описаниями.
- SEO-тексты внизу с упоминанием оборудования (MS JP4 Evo) и брендов (Adidas, Nike, Puma) — хороший social proof.
- "Нам доверяют и рекомендуют более 16 лет" — хорошее доверительное сообщение.
- Подписка на рассылку: "500 Р в подарок за подписку" — хороший лид-магнит.
- Отзывы — ссылка "Посмотреть все отзывы", но не на главной.
- **КРИТИКА:** Элементы доверия есть, но утоплены в нижнюю часть страницы. Чат-бот (желтый кружок) — типовой, без персонализации. Нет калькулятора, нет быстрого оформления.

## 4. GIFTS.RU (Проект 111) — B2B лидер

### Hero-секция
- **Баннер:** "GIFTS.RU — Федеральный поставщик промопродукции и подарков" + CTA "СТАТЬ ДИЛЕРОМ" (зеленая кнопка).
- Фото: предметная съемка (сумки, термокружки, powerbank) — качественная, на сером фоне.
- Ниже: лента новостей с датами — "Весна в новом цвете", "Искусство награждать", кейсы.
- Далее крупно: "Мерч и корпоративные подарки. Поставка со склада и производство".
- **КРИТИКА:** Для B2B — неплохо. Но заголовок "Федеральный поставщик" — слишком формальный, не вызывает эмоции. CTA "СТАТЬ ДИЛЕРОМ" — специфичный, не для конечного заказчика. Нет ценностного предложения "почему именно мы". Фото качественные, но визуал баннера скучноватый — серый фон, корпоративный стиль.

### Навигация
- Верхняя: Лого Проект 111 / Телефон 8-800 / Компания / Производство / Помощь / Войти.
- Вторая строка: Каталог / Фильтр / Поиск / Контакты / Корзина.
- **КРИТИКА:** Двойная шапка — стандартный паттерн B2B. "Фильтр" как отдельный пункт — необычно и неочевидно. Иконки с иконочным шрифтом зеленого цвета. Навигация чистая, но без каталожного дерева в шапке — нужно кликнуть "Каталог" чтобы увидеть категории.

### Каталог категорий
- Иконочная сетка 4 колонки: Одежда / Посуда / Ручки / Сумки / Зонты / Элементы кастомизации / Электроника / Подарочные наборы / Упаковка / Дом / Отдых / Вкусные подарки.
- Под каждой иконкой — 2-3 подкатегории текстом.
- **КРИТИКА:** Иконки линейные, зеленого цвета — узнаваемый стиль, но слишком аскетично. Нет фото товаров в категориях. Подкатегории серым цветом — слабо заметны. Для B2B каталога это работает, но не вдохновляет.

### Визуальное качество
- **Типографика:** Чистая, профессиональная. Хорошие размеры.
- **Цветовая палитра:** Белый + зеленый (фирменный). Минимализм.
- **Whitespace:** Хорошо — страница дышит.
- **Уровень: ПРОФЕССИОНАЛЬНЫЙ, но СКУЧНЫЙ.** Хорошо для B2B-каталога, плохо для вовлечения.

### Доверие и конверсия
- Телефон 8-800 — на видном месте.
- Лента новостей/кейсов — актуальные, с датами.
- "Компания" и "Производство" в навигации — правильный ход для B2B.
- **КРИТИКА:** Нет цифр (сколько клиентов, сколько лет на рынке, объем склада) на первом экране. Нет логотипов клиентов. CTA "Стать дилером" — узкий, отсекает обычных заказчиков. Нет калькулятора нанесения.

## 5. INARI.PRO — B2B корп. мерч, флаги, сувениры

### Hero-секция
- **Баннер-слайдер:** "Весенний коллапс: ПРЕМИАЛЬНЫЕ ФЛАГШТОКИ из нержавейки по цене алюминия!" + цена 24 829 (зачеркнута 35 470).
- Маскот: рыжий лисенок в фирменной повязке "ИНАРИ" — узнаваемый, яркий.
- **КРИТИКА:** Маскот — хорошая идея, добавляет характер. Но сам баннер перегружен: одновременно цена, скидка, маскот, фото флагштоков, текст акции. "Весенний коллапс" — странная метафора для акции, коннотация негативная. Фокус на флагштоках — слишком нишевый для hero.

### Навигация
- Верхняя строка: Поиск (с фильтром "Везде") / Телефон 8-800.
- Основная: Лого "ИНАРИ — Флаги. Мерч. Сувениры" / Каталог / О компании / Свой дизайн / Для клиентов / Контакты / Сравнение / Избранное / Корзина / Войти.
- **КРИТИКА:** Слоган "Флаги. Мерч. Сувениры" — хорошо, сразу понятен бизнес. Но навигация перегружена иконками. Поиск с dropdown "Везде" — хороший паттерн для большого каталога. 26 городов-филиалов в футере — мощный сигнал доверия.

### Каталог / товары
- Секция "Обширный выбор флажной продукции" — горизонтальный скролл с круглыми фото: Флаги триколор / Флаги фирменные / Виндеры / Подставки / Гербы.
- "Мерч который в тренде" — Термостаканы / Бутылки / Рюкзаки / Ремувки / Шопперы.
- "Популярные товары" — карточки с множественными фото (hover-переключение), бейджи "Хит".
- Баннеры категорий: "Стильные фартуки", "Рюкзаки" — яркие фото с градиентными изделиями.
- **КРИТИКА:** Карточки товаров неплохие, но цены не показаны на главной — антипаттерн для B2B (клиент хочет быстро понять ценовой уровень). Баннеры с градиентными товарами — визуально приятные, современные. Смешение флагов и мерча создает размытое позиционирование.

### Визуальное качество
- **Типографика:** Чистая, современная. Хорошие заголовки.
- **Цветовая палитра:** Белый + красный/оранжевый. Яркая, энергичная.
- **Whitespace:** Хорошо.
- **Уровень: ВЫШЕ СРЕДНЕГО.** Современный подход, маскот добавляет уникальности.

### Доверие и конверсия
- Секция "Бесплатные сервисы": от 1 штуки / Бесплатная доставка / Онлайн-оплата / Помощь менеджеров — хорошие доверительные блоки.
- 26 филиалов с телефонами — мощный сигнал масштаба.
- **КРИТИКА:** Тексты сервисов слишком длинные и generic ("Наслаждайтесь удобством бесплатной доставки...") — маркетинговая вата. Нет конкретных цифр, нет отзывов на главной, нет логотипов клиентов.

## 6. PINQ.RU — B2B IT-корп. мерч

### Hero-секция
- **Заголовок:** "Текстильный мерч под задачи бизнеса" — ясный, профессиональный. Хорошая формулировка.
- **Подзаголовок:** "PINQ — это культура производства мерча, помогающая брендам выстраивать эффективные коммуникации." — позиционирование на уровне культуры, не просто продукт.
- **CTA:** "Хочу классный мерч" — нестандартный, дружелюбный текст. Фиолетовая кнопка на розовом фоне.
- **Визуал:** Иллюстрация — группа стилизованных людей, flat-дизайн — приятная, но generic (похожа на Freepik/Dribble стоки).
- **КРИТИКА:** Hero хорошо структурирован. Но фон лавандовый/розоватый — немного "женский", для B2B IT-мерча может не попасть в ЦА. CTA "Хочу классный мерч" — хорошая идея, но контраст фиолетовой кнопки на лавандовом фоне слабый. Иллюстрация — стандартная, не уникальная. Нет конкретики (цены, сроки, тиражи).

### Навигация
- Минималистичная: PINQ (лого) / КАТАЛОГ / НАШИ РАБОТЫ / ЭКСПРЕСС МЕРЧ / КОНТАКТЫ / Телефон.
- 5 пунктов — идеально.
- "ЭКСПРЕСС МЕРЧ" с иконкой бриллианта — акцент на срочный сервис.

## КЛЮЧЕВЫЕ ВЫВОДЫ ДЛЯ НАШЕГО ПРОЕКТА

### Общие слабости рынка (возможности для нас):

1. **Ни один сайт не имеет по-настоящему современного дизайна уровня 2025-2026.** Pinq ближе всех, но и он использует generic-иллюстрации. Возможность: выйти с дизайном уровня международных DTC-брендов.

2. **Конверсионная архитектура слабая у всех.** Нет калькуляторов цен, нет визуальных конфигураторов, нет пошаговых wizards заказа. Возможность: интерактивный конфигуратор мерча.

3. **Social proof недоиспользован.** Только Doneprint показывает реальные кейсы с брендами на главной. Остальные прячут отзывы. Возможность: секция кейсов с логотипами клиентов на первом экране.

4. **B2C-сегмент застрял в 2018.** Printbar и Printio визуально устарели. Vsemayki — рабочий, но без изюминки. Возможность: если у нас есть B2C-компонент — просто сделать красиво уже будет конкурентным преимуществом.

5. **Мобильная версия — "адаптивная по умолчанию" у всех.** Никто не проектировал mobile-first. Возможность: mobile-first подход с тач-жестами, быстрой загрузкой, нативным опытом.

### Лучшие практики для заимствования:

|---|---|---|

### Антипаттерны для избежания:

1. "Вечная скидка -40% на все" (Printbar) — разрушает доверие
2. Двойная перегруженная шапка (Printbar, Printio) — когнитивная перегрузка
3. Эмодзи в title страницы (Printio) — непрофессионально
4. SEO-текстовые портянки внизу страницы (Printbar, Vsemayki) — вредят UX
5. Generic иллюстрации из стоков (Pinq, Printio) — не создают уникальности
6. Outlined CTA-кнопки в hero (Doneprint) — слабый призыв к действию
7. Чат-виджеты, перекрывающие контент (Doneprint) — раздражают
8. Отсутствие цен на B2B-сайтах (Pinq, Doneprint) — барьер для входа
9. Слишком длинный одностраничник (Doneprint) — теряется фокус
10. Ссылка на Instagram для РФ-аудитории (Doneprint) — заблокирован
