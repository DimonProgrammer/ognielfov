## ОБЗОР РЫНКА

### Контекст и размер рынка

Рынок кастомного мерча и брендированной продукции в России — один из немногих сегментов, демонстрирующих устойчивый рост в 2024–2026 гг:

- Объём российского рынка онлайн-торговли в H1 2025 вырос на **36%**, достигнув 5,3 трлн руб. *(источник: Data Insight)*
- Мировой рынок цифровой текстильной печати — **$6,9 млрд к 2027 году** *(прогноз: Grand View Research, требует верификации для РФ-доли)*
- Корпоративный мерч в 2025–2026 гг. — полноценный инструмент брендинга, HR-стратегии и ESG
- **Число продавцов** на WB/Ozon, занимающихся мерчем и кастомной одеждой, значительно выросло в 2024–2025 *(точная цифра «удвоения» требует верификации через актуальную аналитику маркетплейсов)*

### Ключевые тренды 2025–2026

| Тренд | Описание |
|-------|----------|
| Глубокая кастомизация | Ремувки, пины, патчи, пуллеры — быстрый апгрейд базы до уникального мерча |
| Mix Print | Комбинации технологий нанесения, текстур, рельефа и отражения света |
| Эко-материалы | PET-переработка, органический хлопок, конопля |
| Мерчшопы | Корпоративные магазины мерча как программы лояльности |
| Минимализм + All-over | Два полюса: гигантские принты и минималистичные акценты |
| Русский орнамент | Этнические мотивы + современная графика |
| Персонализация | Именные надписи, иллюстрации под отдел/проект/событие |

### Целевые сегменты рынка (для нашего проекта)

- **B2C**: кастомные заказы для физлиц (подарки, самовыражение, фандом)
- **Продавцы WB/Ozon**: производство товаров для перепродажи с маркировкой «Честный знак»
- **Корпоративные клиенты**: брендирование, мерч для сотрудников, клиентов, мероприятий

### 1. Vsemayki.ru (ВсеМайки)

**URL:** https://www.vsemayki.ru/
**Основана:** 2007

#### Целевая аудитория
Широкая B2C-аудитория: физлица, ищущие готовые принты или хотящие создать собственный дизайн. Частные покупатели подарков, фанаты брендов, молодёжь.

#### Каналы привлечения
- **SEO** — мощная органика по брендовым и категорийным запросам
- **Контекстная реклама** — Яндекс.Директ
- **Мобильное приложение** — App Store (активно используется)
- **Маркетплейсы** — присутствие на Ozon

#### Ключевые рекламные сообщения / УТП
- «Более 150 000 дизайнов в наличии»
- «Лучший конструктор футболок с принтами на заказ»
- «Онлайн-конструктор одежды»
- Позиционирование: огромный выбор + возможность создать своё

#### Ассортимент и специализация
Футболки, толстовки, худи, кружки, кепки, аксессуары, сувениры. Специализация — B2C розница, готовые принты + кастомный конструктор.

#### Ценовая структура
Средний ценовой сегмент, футболки от ~700–1500 руб. Постоянные акции и скидки.

#### Что клиенты любят
- Огромный выбор готовых дизайнов
- Удобный онлайн-конструктор
- Яркие, чёткие принты
- Приятная, мягкая ткань (premium-линейка)
- Оперативная поддержка клиентов

#### Что клиенты не любят
- **Непрозрачные сроки** доставки (ждали 7+ дней вместо обещанных 2)
- Часть SKU — тонкая синтетика вместо хлопка
- Принт бледнеет после первой стирки (не у всех)
- Сложная рекламационная работа — «попрощайтесь с деньгами»
- Ухудшение клиентского сервиса в последние годы

#### Качество и структура сайта: **8/10**
- **Хорошо:** Мощный онлайн-конструктор, огромный каталог, удобная навигация, мобильное приложение, визуализация результата в реальном времени
- **Плохо:** Перегруженность каталога, слабая работа с B2B-сегментом, нет чёткого раздела для оптовых/корпоративных заказов

#### SWOT

| Сильные стороны | Слабые стороны |
|-----------------|----------------|
| Лидер рынка, 100K+ посетителей/день | Качество нестабильное |
| 500K+ заказов/год | Слабый B2B |
| 150K+ дизайнов | Долгая доставка |
| Мобильное приложение | Плохая рекламационная работа |
| Мощный конструктор | Синтетика вместо хлопка в дешёвых SKU |

## РЕКОМЕНДАЦИИ ПО ДИФФЕРЕНЦИАЦИИ

### Стратегия 1: «Три окна — одна фабрика»

Разделить сайт на три чётких входа для разных ЦА:
- **Окно 1 — B2C:** «Хочу мерч для себя / подарок» → конструктор, быстро, онлайн
- **Окно 2 — Маркетплейсы:** «Продаю на WB/Ozon» → под ключ с маркировкой и отгрузкой
- **Окно 3 — Бизнес:** «Корпоративный заказ» → калькулятор, тираж, логотип, кейсы

**Почему работает:** Ни один конкурент не работает одновременно во всех трёх нишах с чётким позиционированием.

---

### Стратегия 2: Прозрачное ценообразование как УТП

Внедрить **онлайн-калькулятор** с публичными ценами даже для B2B:
- Тираж → материал → вид нанесения → мгновенная цена
- Это разрушает главный барьер B2B-рынка: «позвоните нам»
- Конвертирует холодный трафик в реальные заказы без менеджера

---

### Стратегия 3: Технологии как образование, не как список

Вместо «мы делаем DTF, шелкографию, сублимацию» — создать страницу-гид:
- «Когда нужна шелкография? Когда DTF? Когда УФ?»
- Образовательный контент снимает барьеры принятия решения
- Клиент приходит к нам образованным → меньше согласований → выше LTV

---

### Стратегия 4: WB/Ozon-специализация — полный цикл

Разработать отдельный лендинг/раздел «Мерч для маркетплейсов»:
- Производство партии
- Маркировка «Честный знак»
- Фото для карточки товара (на белом фоне + lifestyle)
- Упаковка под стандарты WB/Ozon
- Отгрузка на склад (самовывоз / доставка)

**Это закрывает всю боль** продавца маркетплейса в одном месте.

---

### Стратегия 5: Репутация через прозрачность

Главные жалобы на конкурентов: непрозрачные сроки, рекламации игнорируются, качество нестабильное.

Дифференциация:
- **Гарантия срока** (не успели — скидка/компенсация)
- **Образец перед тиражом** как стандартная услуга
- **Личный кабинет** с онлайн-статусом заказа
- **Открытые отзывы** с ответами

---

## ТОП-3 ИНСАЙТА ДЛЯ ПОЗИЦИОНИРОВАНИЯ САЙТА

### Инсайт #1: Рынок фрагментирован — выиграет тот, кто охватит все три сегмента

**Ситуация:** B2C-игроки (Vsemayki, PrintBar) не работают с B2B. B2B-игроки (INARI, PINQ, Проект 111) не работают с B2C и маркетплейсами. DonePrint работает с маркетплейсами, но не с B2C.

**Инсайт:** Первый игрок, который создаст **удобный мультиканальный сайт** с чётким разделением для трёх ЦА при едином производстве — займёт уникальную позицию на рынке без прямых конкурентов.

**Применение:** Архитектура сайта должна начинаться с выбора «Кто вы?» и вести каждую ЦА по своему пути с релевантным предложением.

---

### Инсайт #2: Доверие — главный дефицит рынка

**Ситуация:** Анализ отзывов всех конкурентов показывает: клиенты уходят не из-за цены, а из-за **непрозрачности** — непонятных сроков, скрытых цен, игнорирования рекламаций, нестабильного качества.

**Инсайт:** Компания, которая **первой сделает доверие своим главным активом** (публичные цены, гарантия срока, образцы, личный кабинет с трекингом, открытые отзывы) — автоматически выиграет конкурентную борьбу в любом из трёх сегментов.

**Применение:** На сайте должны быть видны: калькулятор цен, гарантии, сроки, реальные отзывы с ответами компании, FAQ о рекламациях.

---

### Инсайт #3: Продавцы WB/Ozon — самый быстрорастущий и незакрытый сегмент

**Ситуация:** Число продавцов на маркетплейсах удвоилось за год. Комиссии растут (до 35–47%), продавцы ищут конкурентное преимущество в **уникальном продукте**. Кастомный мерч с печатью — один из способов выделиться. При этом DonePrint и Мам Пошей закрывают только часть потребностей этой ЦА.

**Инсайт:** Создание **отдельного продукта «для продавцов WB/Ozon»** — производство + маркировка + фото + отгрузка — это голубой океан с огромной платёжеспособной аудиторией и минимальной конкуренцией.

**Применение:** Отдельный лендинг «Мерч для маркетплейсов», SEO под запросы «производство одежды для WB», «маркировка мерча Честный знак», «пошив под Wildberries», таргет на сообщества продавцов.
