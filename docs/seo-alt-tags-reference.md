# Справочник alt-тегов для изображений elfprint.ru

> Подготовлено для SEO-оптимизации при замене placeholder'ов на реальные фото.
> При вставке `<img>` использовать alt из этого файла.

---

## Правила составления alt-тегов

1. **Описывать что на фото** — не ключевые слова, а конкретику
2. **Включать бренд** — "Огни Эльфов" в ключевых фото (hero, портфолио)
3. **Указывать метод печати** — если виден на фото
4. **Указывать изделие** — тип товара
5. **Не спамить** — 5-15 слов, без перечисления ключевиков
6. **На русском** — основной язык сайта
