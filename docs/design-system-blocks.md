# Дизайн-система: блоки и принципы

## Принципы построения страниц

### 1. Ритм секций — чередование паттернов
Никогда не ставить два одинаковых паттерна подряд. Чередовать:
- **Фон:** белый (#FFFFFF) / серый (#F5F5F7) / тёмный (#1D1D1F) / акцент (#C6F24E)
- **Ширина контента:** contained (1280px) / full-bleed (100%) / mixed
- **Сетка:** bento (разный размер карточек) / equal grid / split layout / single column
- **Взаимодействие:** статика / анимация при скролле / marquee / горизонтальный скролл

### 2. Визуальная иерархия через размер
- Главный элемент секции всегда визуально крупнее (span-2, больше padding, другой фон)
- Приоритетный контент = больше пространства (корпоративный сегмент > остальные)
- Никогда не делать все карточки одинаковыми, если у контента есть иерархия

### 3. Каждая секция — уникальный ритм
Не использовать один и тот же grid на соседних секциях. Если каталог = 3x2, то следующая секция НЕ должна быть 3x2.

## Переиспользуемые блоки (компоненты)

### A. Navigation (sticky frosted glass)
**Используется на:** всех страницах
- Sticky top, backdrop-filter blur(20px)
- Логотип + nav links + телефон + мессенджеры + CTA кнопка
- CSS класс: `.nav-glass`

### B. Stats Strip (full-width lime)
**Используется на:** главная, о компании, корпоративным клиентам
- Full-width фон #C6F24E
- 3-5 метрик с разделителями
- Animated countUp при скролле
- CSS класс: `.stats-strip`, JS: `.count-up[data-target]`

### C. Bento Grid (1 big + N small)
**Используется на:** главная (услуги), страницы услуг, каталог
- CSS Grid с одной карточкой span-2 (row или col)
- Остальные карточки стандартного размера
- Большая карточка = главный элемент секции

### D. Asymmetric Cards (2/3 + 1/3)
**Используется на:** главная (аудитории), маркетплейсы, корпоративным клиентам
- Приоритетная карточка col-span-2, тёмный фон (#1D1D1F)
- Вторичные карточки стопкой справа, светлый фон
- Лайм-чипы для ключевых слов

### E. Stepper (process steps)
**Используется на:** главная, услуги, маркетплейсы
- Горизонтальный (desktop) / вертикальный (mobile) ряд шагов
- Кружки с номерами, соединённые линиями
- Линии заполняются цветом при скролле
- CSS класс: `.step-connector`

### F. Product Cards Grid (featured + grid)
**Используется на:** главная (каталог), каталог (хаб), категории
- Одна featured-карточка span 2 cols + 2 rows
- Остальные карточки обычного размера
- Обязательно: фото-зона + название + цена "от X руб."

### G. Case Card — Featured + Grid
**Используется на:** главная, портфолио
- 1 featured кейс: горизонтальный layout (фото 50% + текст 50%)
- Остальные кейсы: вертикальные карточки в ряд
- Бейдж отрасли (лайм), теги технологий

### H. Technology Card (photo + CTA)
**Используется на:** главная, страницы услуг
- Цветная фото-зона сверху (gradient bg + иконка/фото)
- Бейдж минимального тиража (лайм pill)
- Описание + заметная pill-кнопка перехода
- Сетка 3x2

### I. Testimonials Marquee
**Используется на:** главная, о компании
- 2 ряда карточек, infinite CSS marquee (противоположные направления)
- Pause on hover
- Yandex badge (4.9 звёзд) рядом с заголовком
- Ссылка на Яндекс.Карты

### J. FAQ Two-Column Split
**Используется на:** главная, отдельные страницы услуг
- Left (40%): sticky заголовок + описание + ссылка "напишите нам"
- Right (60%): accordion
- Фон #F5F5F7

### K. CTA Dark Panel
**Используется на:** все страницы (финальная секция перед footer)
- Фон #1D1D1F, белый текст
- Left: заголовок + описание + кнопки мессенджеров
- Right: белая form card с тенью (имя, телефон, submit)
- Consent checkbox обязателен (152-ФЗ)

### L. Footer
**Используется на:** всех страницах
- 4 колонки: О компании / Навигация / Контакты / Карта
- ИНН/ОГРН обязательно (Яндекс коммерческие факторы)
- Ссылки на legal-страницы

### M. Trust Logos Marquee
**Используется на:** главная (hero), о компании, корпоративным клиентам
- Infinite CSS marquee с логотипами клиентов
- Grayscale, opacity 25%
- Дублирование для бесшовного цикла

### N. Chips / Tags
**Используется на:** везде
- Pill-shape (border-radius: 50px)
- Фон #F5F5F7, текст #6E6E73 (стандарт)
- Фон #C6F24E, текст #1D1D1F (акцент)
- Используются для: технологий, тегов кейсов, ключевых слов аудитории

## Правила комбинирования на внутренних страницах

### Страница услуги (например /uslugi/dtf)
1. Hero Split (текст + фото оборудования)
2. Stats Strip (характеристики технологии)
3. Bento Grid (преимущества)
4. Product Cards (что можно печатать)
5. Case Cards (кейсы с этой технологией)
6. FAQ Two-Column
7. CTA Dark Panel

### Страница каталога (/catalog)
1. Заголовок + описание
2. Product Cards Grid (featured + grid)
3. CTA Dark Panel

### Корпоративным клиентам (/korporativnym-klientam)
1. Hero Split
2. Stats Strip
3. Stepper (процесс работы)
4. Case Cards (корпоративные кейсы)
5. Testimonials Marquee
6. FAQ Two-Column
7. CTA Dark Panel

### Маркетплейсы (/marketplejsy)
1. Hero Split
2. Stats Strip (300 изд./день, 4 клиента)
3. Stepper (полный цикл FBO/FBS)
4. Technology Cards (доступные методы)
5. FAQ Two-Column
6. CTA Dark Panel
