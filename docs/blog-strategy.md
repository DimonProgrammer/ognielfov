## 1. Целевые Аудитории и Темы

### Сегмент 1: B2C (физические лица, стартапы, малый бизнес)
**Проблема:** Не знают, как выбрать технологию печати, страшатся цен, сомневаются в качестве
**Решение:** Статьи с прямыми рекомендациями и ценами

**Проблема:** Нужны гайды по планированию мерча, бюджетированию, примеры из реальной практики
**Решение:** Статьи о процессе, кейсах, best practices

**Проблема:** Нужны данные о спросе, маржах, популярных товарах, логистике
**Решение:** Статьи для оптовиков и перепродавцов

## 3. Общие SEO-требования

- Каждая статья: 1500+ слов, 2+ изображений (placeholder)
- Заголовки: H1 (ровно 1), H2 (2-4), H3 по необходимости
- Table of Contents обязателен
- Внутренние ссылки: минимум 3 ссылки на другие страницы сайта
- Кода ссылок: на услуги (uslugi.html, uslugi-dtf.html), каталог (catalog.html)
- External ссылки: только на авторитетные источники (если есть)
- Дата публикации и время чтения в метаданных
- Schema.org: Article + BreadcrumbList

## 4. Жанровые правила для статей

✅ **Делать:**
- Конкретные примеры с цифрами
- Сравнительные таблицы
- Реальные кейсы из портфолио
- Пошаговые гайды
- Q&A секции
- CTA в каждой статье (но естественно, не навязчиво)

❌ **НЕ делать:**
- "рубленые фразы" ("Выбрал. Заказал. Получил.")
- Кинематографический сценарий ("Понедельник, 9 утра...")
- Пустые связки ("Важно понимать...", "Давайте разберёмся...")
- Выдуманные истории (всё должно быть основано на реальном опыте компании)
- AI-клише (спойлеры, plot twists, fake emotion)
